# 2D-Shooter
A 2d testproject as preparation for next gamejam.


<b>Controls</b> <br>
Movement: WASD or Arrow keys <br>
Shooting: Space or left click<br>
Grenade: Left ctrl or right click<br>

![outside area](screenshots/shooter.png)
Tipp: Shoot crates and toilets to get ammunition and health. <br>

![inside area](screenshots/shooter2.png)
Currently there is no game over screen so the game never ends.
